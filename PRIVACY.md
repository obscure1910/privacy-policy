# Datenschutzerklärung
## Privater Nutzungsbereich
Diese n8n Anwendung wird ausschließlich für persönliche, private Zwecke genutzt.
## Verarbeitete Daten
1. **Eigene Google Drive Dokumente**: Nur von mir ausgewählte Dateien werden verarbeitet
2. **Keine personenbezogenen Daten anderer**: Es werden keine fremden personenbezogenen Daten verarbeitet
3. **Temporäre Verarbeitung**: Daten werden nur während der Verarbeitung im Arbeitsspeicher gehalten
## Zweck der Verarbeitung
- Extraktion von Dokumenteninhalten für KI-gestützte Suche
- Erstellung von Vektoren für semantische Suche
- Temporäre Speicherung während der Verarbeitung
## Speicherung
- **Keine dauerhafte Speicherung**: Dokumentinhalte werden nach Verarbeitung nicht gespeichert
- **Lokale Ausführung**: Alle Prozesse laufen auf meinem eigenen Server
## Datenweitergabe
- **Keine Weitergabe**: Daten werden nicht an Dritte weitergegeben
- **Nur benötigte APIs**: Ausschließlich Google Drive API für den eigenen Dateizugriff
## Meine Rechte
Als einziger Betroffener habe ich jederzeit Zugriff auf alle verarbeiteten Daten.
